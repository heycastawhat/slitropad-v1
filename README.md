# Features
It Has 6 Keys.
Key 1: Play/Pause
Key 2: Skip Forward/Back
Key 3: Slack/Discord Button
Key 4: Apple Music Button
Key 5: Push To Mute 
Key 6: Mission Control
(Its made for a mac, all buttons that open apps take advantage of spotlight)

# BOM
- 1x 3D Printed Case
- 6x Blank DSA keycaps
- 6x MX-Style switches
- 4x M3x16mm screws
- 4x M3x5mx4mm heatset inserts
